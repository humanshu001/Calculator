Calculator JavaScript Code Documentation

Variables:

  • runningTotal: Stores the result of calculations.
  • buffer: Stores the current input.
  • previousOperator: Stores the previous operator used.
  • maxBufferLength: Specifies the maximum allowed length for the input buffer.

DOM Elements:

  • screen: Reference to the screen element where input and results are displayed.
  • buttons: Reference to all calculator buttons.

Function: buttonClick(value)

  • Handles button clicks.
  • Determines whether the clicked value is a number or a symbol and delegates to the appropriate function.

Function: handleSymbol(symbol)

  • Handles symbol inputs such as "C", "=", "‚Üê", "+", "-", "*", "/".
  • Clears the calculator, calculates the result, handles backspace, and performs mathematical operations.

Function: handleMath(symbol)

  • Handles mathematical operations.
  • Converts the current buffer to an integer and updates the running total based on the previous operator.

Function: flushOperation(intBuffer)

  • Performs the actual mathematical operation based on the previous operator.

Function: handleNumber(numberString)

  • Handles numeric inputs.
  • Appends the numeric input to the buffer, considering the maximum buffer length.

Function: updateScreen()

  • Updates the screen with the current buffer content while preserving styling.
  • Displays only the last 10 digits if the number is too long.

Function: init()

  • Initializes the calculator by adding click event listeners to buttons.

Initialization:

  • Adds a DOMContentLoaded event listener to ensure the calculator is initialized after the page loads.
